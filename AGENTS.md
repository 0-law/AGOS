# AGENTS.md

## 项目定位

本项目是 A股短线题材复盘与个股逻辑提取系统，当前标准为 `AGOS v2.2`。

输出文件：

1. `输出结果/Markdown研报/01_日内投研研报_YYYY-MM-DD_HHMMSS.md`
2. `输出结果/个股总表/02_个股逻辑提取表_YYYY-MM-DD_HHMMSS.csv`
3. `输出结果/HTML研报/03_日内投研研报_YYYY-MM-DD_HHMMSS.html`

## 路由规则

### 日常复盘任务

读取：

1. `规则库/AGOS_v2.2_每日执行提示词.md`
2. `规则库/AGOS_v2.2_CSV字段规范.md`
3. `规则库/AGOS_v2.2_输出校验器.md`

### 修改系统规则任务

读取：

1. `规则库/AGOS_v2.2_完整规范.md`
2. `规则库/AGOS_v2.2_每日执行提示词.md`
3. `规则库/AGOS_v2.2_输出校验器.md`
4. `规则库/AGOS_v2.2_CSV字段规范.md`

### 输出校验任务

读取：

1. `规则库/AGOS_v2.2_输出校验器.md`
2. `规则库/AGOS_v2.2_CSV字段规范.md`
3. 必要时读取 `规则库/AGOS_v2.2_完整规范.md`

## 工程边界

- 不输出买卖建议
- 不输出目标价
- 不输出仓位建议
- 不编造代码
- 不编造业务
- 不把小作文当事实

## 目录约定

- `规则库/`：放 AGOS 规范、每日执行提示词、校验器、CSV 字段规范。
- `输入资料/`：放原始市场资料。
- `输出结果/Markdown研报/`：放 Markdown 研报。
- `输出结果/个股总表/`：放 CSV 个股总表。
- `输出结果/HTML研报/`：放 HTML 阅读版研报。
- `归档/`：放按日期归档的原始资料与输出结果。
